# Yomiji Website

Landing page for 読路 / Yomiji, an offline-first WaniKani study client.

Yomiji source code: https://github.com/notquite28/yomiji

Built with Vite, React, TypeScript, Tailwind CSS, GSAP ScrollTrigger, and Lenis smooth scrolling.

## Development

```bash
npm run dev
npm run build
npm run lint
```
